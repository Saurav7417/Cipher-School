Binary Search Tree (BST) Implementation in Java
📌 Project Overview

This project demonstrates the implementation of a Binary Search Tree (BST) using Java.
It focuses on performing fundamental tree operations such as insertion, deletion, and updating nodes while maintaining BST properties.

The project was developed as part of Data Structures and Algorithms learning during training at CipherSchools.

🚀 Features

Insert a node into the Binary Search Tree

Delete a node from the tree

Update an existing node value

Maintain BST ordering property

Console-based implementation

🛠️ Technologies Used

Java

Object-Oriented Programming (OOP)

Data Structures & Algorithms

📂 Operations Implemented
1️⃣ Insertion

Adds a new node at the correct position following BST rules.

2️⃣ Deletion

Removes a node handling:

Leaf node deletion

Node with one child

Node with two children

3️⃣ Updation

Updates node value while preserving BST structure.
